# TingHu 推文剪藏收件箱(路线A)

在手机 GitHub App 或网页上编辑本文件,把推文粘贴到下方粘贴区,保存(commit)即可。
Action 会自动把新条目规范化进 `tinghu/tweets.jsonl`,然后清空粘贴区。

一条的格式(顺序随意,格式宽松):

```
2026-07-08 21:30        ← 时间行,可省略(省略按剪藏当天算)
推文正文,可多行
https://x.com/TingHu888/status/1234567890
```

多条之间用一行 `---` 隔开。没有链接的条目会被保留在此等你补链接。
